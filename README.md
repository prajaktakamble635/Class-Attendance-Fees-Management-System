# Class Attendance & Fees Management System

A Full Stack Class Attendance & Fees Management System developed using React.js, Node.js, Express.js, MySQL, and Sequelize ORM.

## 🚀 Features

- User Login & Authentication
- Dashboard
- Student Management
- Class Management
- Attendance Management
- Fee Collection
- Pending Fees Report
- Payment History
- Exam Management
- Result Management
- Staff Management
- Reports & Analytics

## 👥 User Roles

- Admin
- Staff
- Student

## 🛠️ Tech Stack

### Frontend
- React.js
- Material UI
- Tailwind CSS
- Axios

### Backend
- Node.js
- Express.js
- Sequelize ORM

### Database
- MySQL

### Authentication
- JWT (JSON Web Token)

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/prajaktakamble635/Class-Attendance-Fees-Management-System.git
```

### Install Frontend

```bash
cd client
npm install
npm run dev
```

### Install Backend

```bash
cd server
npm install
npm start
```
